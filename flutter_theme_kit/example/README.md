# Flutter Theme Kit Example

This example demonstrates how to use `app_theme_kit` in a real Flutter app.

## Features Demonstrated

- **Light and Dark themes** with `ThemeMode.system`
- **Typography**: Display, Title, and Body text styles using Google Fonts
- **CardTheme**: Rounded cards with elevation and consistent spacing
- **TabBarTheme**: Styled tabs in the AppBar
- **NavigationBarTheme**: Material 3 bottom navigation bar
- **BottomAppBarTheme**: Styled bottom bar with consistent colors
- **AppSpacing**: Standardized spacing and radius tokens

## How to Run

```bash
cd example
flutter run
